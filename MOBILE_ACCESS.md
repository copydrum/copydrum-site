# 모바일에서 로컬 개발 서버 접근하기

## 📱 모바일 접근 방법

### 1. PC와 모바일이 같은 Wi-Fi 네트워크에 연결되어 있는지 확인

- PC와 모바일 기기가 **같은 Wi-Fi 네트워크**에 연결되어 있어야 합니다.

### 2. PC의 로컬 IP 주소 확인

#### Windows
```powershell
ipconfig
```
또는
```powershell
ipconfig | findstr /i "IPv4"
```

**현재 PC IP:** `192.168.109.46`

#### Mac/Linux
```bash
ifconfig | grep "inet "
```

### 3. 개발 서버 시작

```bash
npm run dev
```

서버가 시작되면 다음 주소들이 표시됩니다:
- `http://localhost:3000` (PC에서만 접근)
- `http://192.168.109.46:3000` (모바일에서 접근)

### 4. 모바일 브라우저에서 접근

모바일 기기의 브라우저에서 다음 주소로 접근하세요:

```
http://192.168.109.46:3000
```

**중요:** IP 주소는 PC가 재연결될 때마다 변경될 수 있습니다. IP가 변경되면 다시 확인해야 합니다.

## 🔧 설정 확인

`vite.config.ts` 파일에 다음 설정이 되어 있어야 합니다:

```typescript
server: {
  port: 3000,
  host: '0.0.0.0',  // 네트워크 접근 허용
}
```

✅ 현재 설정이 올바르게 되어 있습니다.

## 🚨 문제 해결

### 모바일에서 접속이 안 될 때

1. **방화벽 확인**
   - Windows 방화벽에서 포트 3000 허용 필요할 수 있습니다
   - 제어판 > Windows Defender 방화벽 > 고급 설정 > 인바운드 규칙

2. **IP 주소 재확인**
   - PC의 IP 주소가 변경되었을 수 있습니다
   - 위의 명령어로 다시 확인하세요

3. **같은 네트워크인지 확인**
   - PC와 모바일이 같은 Wi-Fi에 연결되어 있는지 확인
   - 모바일 데이터가 아닌 Wi-Fi를 사용해야 합니다

4. **개발 서버 재시작**
   ```bash
   # Ctrl + C로 서버 중지 후
   npm run dev
   ```

## 📝 참고

- 개발 서버를 중지하면 모바일에서도 접근할 수 없습니다
- 코드 변경 시 모바일에서도 자동으로 반영됩니다 (Hot Module Replacement)
